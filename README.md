# GetHashtags
Easy to use with GUI and reporting tool. Get twits from hashtags.

Export file options :
CSV
PDF
HTML
MHT
RTF
XLS
XLSX
CSV
TEXT FILE
IMAGE FILES

1-Enter a hashtag for search.
2-Confirm a date.
3-Enter a twit count.
4-Press Scan.
5-Get a report.
